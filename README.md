# Проектная работа «Посмотри в окно» 

в проекте **Проектная работа «Посмотри в окно»** были реазированы на практике новые теоритические знания такие как:

* доступность

* псевдоклассы

* вес в css и их общая значимость в коде

* автоматизация и др.

Данный проект направлен на закрепление перечисленных выше знаний, а так же направлен на то, чтобы сделать реализовать цифровую доступность на данном сайте и отличия второго проекта от первого заключаются в том, что в первом преокте нам необходимо было понять основы вертски, здесь мы более углубленно продолжаем заниматься ею во втором проекте.

## Системные требования:

* **Node.js** v18.x или выше
*(проверьте: node -v)*

* **Браузер:** Chrome/Firefox/Safari/Edge последней версии

## Установка и запуск

1. **Клонируйте репозиторий:**

```bash
git clone https://ваш-репозиторий.git
cd название-проекта
```

2. Не забудьте установить зависимости npm i

## Команды для тестирования

**prettier:**

1. проверка

```bash
npm run check
```

2. исправить описанными методами

```bash
npm run format
```

**stylelint:**

1. вывод всех ошибок в строках

```bash
npm run link
```

2. исправить ошибки 

```bash
npm run linkfix
```

## Упрощенные команды

1. проверить по prettier и по stylelint

```bash
checkAll
```

2. исправить по prettier и по stylelint

```bash
fixAll
```
